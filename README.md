# Dartapi
본 프로젝트는 Dart api를 이용해 재무제표를 가져오고 llm을 이용해 등급을 매깁니다.


# 종속성 설치

```
pip install -r requirements.txt
```

## env 설정
.env는 다음과 같이 작성해야 합니다.

```
OPENAI_API_KEY="your-key"
DART_API_KEY="your-key"
```
